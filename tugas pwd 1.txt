PENJELASAN KODE :
<!DOCTYPE html> untuk menentukan tipe dokumen
<HTML> … </HTML> kumpulan karakter yang mengawali dan mengakhiri suatu tag. Karakter yang digunakan (<>). Tag awal disebut opening tag, tag akhir disebut ending tag ditandai dengan (/)
 <HEAD> … </HEAD> Diantaranya adalah area dari heading halaman. Digunakan untuk script/perintah khusus yang tidak mempunyai hubungan dengan format dari halaman. 
<TITLE> … </TITLE> Digunakan untuk pemberian judul
<BODY> … </BODY> digunakan untuk area dimana yang dilihat adalah merupakan tampilan dari web/ isi dari halaman.
 <style> ... </style> Digunakan untuk :  Hal-hal yang berkaitan dengan sebuah tag spesifik HTML,  Mendefinisikan sebuah tag HTML tertentu yang memiliki nama class spesifik dengan nilai –nilai  style sheet tertentu,
 Mendefinisikan sebuah nama identitas spesifik yang memiliki nilai-nilai style sheet tertentu.
<font>...</font> tag untuk membuat font, warna, dan ukuran untuk teks
<link>...</link> tag untuk membuat hubungan antara dokumen dan sumber daya eksternal
<h1> dan <h3> digunakan untuk membuat heading
<header>...</header> tag untuk membuat sebuah header untuk dokumen atau bagian
<nav>...</nav> digunakan untuk membuat navigasi link
<aside> tag untuk membuat konten lain selain dari konten halaman
<img> tag untuk membuat gambar
<table>...</table> tag untuk membuat tabel
<thead>...</thead> tag untuk mengelompokkan isi header dalam sebuah tabel
<ul>...</ul> tag untuk membuat teks yang memiliki gaya yang berbeda dari teks biasa lainnya
<section>...</section> tag untuk membuat bagian dalam dokumen
<article>...</article> tag untuk membuat sebuah artikel
<figure>...</figure> tag untuk menentukan konten mandiri
<!--...--> tag untuk membuat komentar
<hgroup>...</hgroup> tag untuk mengelompokkan elemen heading
<p>...</p> tag untuk membuat paragraf
<th>...</th> tag untuk membuat sebuah sel header tabel
<footer>...</footer> tag untuk membuat footer untuk dokumen atau bagian
<address>...</address> tag untuk membuat kontak alamat
