PENJELASAN KODE:
<!DOCTYPE html> untuk menentukan tipe dokumen
<HTML> … </HTML> kumpulan karakter yang mengawali dan mengakhiri suatu tag. Karakter yang digunakan (<>). Tag awal disebut opening tag, tag akhir disebut ending tag ditandai dengan (/)
<HEAD> … </HEAD> Diantaranya adalah area dari heading halaman. Digunakan untuk script/perintah khusus yang tidak mempunyai hubungan dengan format dari halaman. 
<TITLE> … </TITLE> Digunakan untuk pemberian judul
<link> Digunakan untuk membuat hubungan antara dokumen dan sumber daya eksternal(paling sering digunakan untuk link style sheet)
<script>...</script> Tag untuk membuat script disisi klien
<BODY> … </BODY> digunakan untuk area dimana yang dilihat adalah merupakan tampilan dari web/ isi dari halaman.
 <style> ... </style> Digunakan untuk :  Hal-hal yang berkaitan dengan sebuah tag spesifik HTML,  Mendefinisikan sebuah tag HTML tertentu yang memiliki nama class spesifik dengan nilai –nilai  style sheet tertentu,
 Mendefinisikan sebuah nama identitas spesifik yang memiliki nilai-nilai style sheet tertentu.
<div>...</div> tag untuk membuat sebuah bagian dalam dokumen
<center>...</center> tag untuk membuat jajaran teks menjadi ditengah 
<h1> to <h5> tag untuk membuat heading
<p>...</p> tag untuk membuat paragraf
<input>...</input> tag untuk membuat sebuah kontrol input
<button>...</button> tag untuk membuat sebuah tombol yang dapat di klik
<form>...</form> tag untuk membuat sebuah form html untuk input pengguna
<span>...</span> tag untuk membuat sebuah bagian dalam dokumen
